# # Auto-GTP Generation Webpage for Transformers

This project is a web-based calculator designed to automatically generate the **Guaranteed Technical Particulars (GTP)** for various types of transformers. It streamlines the engineering process by automating complex calculations required for transformer specification and design.

> This tool was custom-built and is specially dedicated to supporting the engineering and design teams at **Mahati Industries**.
